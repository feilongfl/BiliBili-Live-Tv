用于观看B站直播，由于某些原因在电视上看不了，所以自己做了一个，本人不会安卓开发，东拼西凑的代码，不保证通用。

* 本人开发环境环境：

 1. 华为g750-t00（安卓4.4.4）
 2. 天猫魔盒1s plus（一生黑的yunos 6.0.0）（没有机顶盒的亲们千万别买天猫魔盒，谁用谁后悔）
